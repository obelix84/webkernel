Описание некоторых ЗАРЕЗЕРВИРОВАННЫХ названий парметров.

1) source, тип string - параметр в коттором содержится данные о том откуда брать шаблон, например:
    FILE: cms_pattern - шаблон cms_pattern.tmpl берется из фала.

2) count, тип integer - счетчик кол-ва записей в dataSource

3) designer, тип string - содержит название компоненты отвечающей за дизайн, вместе с номер 	конфигурации

4) dataSource, тип string - содержит название компоненты отвечающей за компановку данных, вместе с номером конфигурации
